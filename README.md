# HireLens AI — Resume Analyzer & ATS Optimizer

HireLens AI is a frontend MVP that analyzes resume content against a job description and generates an estimated ATS compatibility score, missing keywords, strengths, weaknesses, and improved resume bullet suggestions.

## Live Demo

https://ankitraj21805-lgtm.github.io/hirelens-ai-resume-analyzer/

## Problem

Many students and junior developers apply with generic resumes that are not aligned with job descriptions. HireLens AI helps users understand what keywords and improvements are missing before applying.

## Features

- Resume text analysis
- Job description comparison
- Estimated ATS compatibility score
- Keyword gap analysis
- Resume strengths and weaknesses
- Improved resume bullet suggestions
- Local browser history
- Responsive dark SaaS-style UI

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## Current Version

This is Version 1 frontend MVP. It uses transparent keyword-based scoring and local browser storage.

## Disclaimer

HireLens AI provides an estimated ATS-style score based on keyword matching, resume structure, and role relevance. It does not guarantee job selection or actual ATS ranking.
